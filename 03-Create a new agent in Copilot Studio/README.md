# 🤖 Step 3 - Create a New Agent in Copilot Studio

> Log in and create a blank agent. You will configure its identity first,
> then build topics and automation on top of it in the steps that follow.

---

## What Are We Doing Here?

In this step you will create the foundation of your IT Help Desk Agent inside Copilot Studio.
Think of this as setting up the agent's **identity and personality** — its name, purpose,
and the instructions that shape how it behaves in every conversation.

---

## 🛠️ Create the Agent

### Step 1 - Open Copilot Studio

Navigate to Copilot Studio using the link below and sign in with your Microsoft 365 account:

🔗 [https://copilotstudio.microsoft.com](https://copilotstudio.microsoft.com)

---

### Step 2 - Create a New Agent

1. In the left panel, click **Create**
2. Select **New agent**
3. Click **Skip to configure** to start with a blank canvas

---

### Step 3 - Configure the Agent Identity

Fill in the agent details as follows:

| Field | Value |
|---|---|
| **Name** | `IT Help Desk Agent` |
| **Description** | `I help employees report IT issues and submit support tickets.` |

---

### Step 4 - Add Instructions

In the **Instructions** field, paste the following:

You are an IT Help Desk assistant. Be professional and concise.
Collect the user's name, email, a short description of their issue,
and the urgency level. Confirm the ticket has been submitted.
Do not attempt to solve technical issues yourself.


---

### Step 5 - Create the Agent

Click **Create** to save and generate your agent.

---

## 💡 Important Note

> The **Instructions** field shapes how the agent's generative AI responds.  
> Keep it short and directive — long instructions tend to dilute behaviour
> and can cause the agent to respond inconsistently.
