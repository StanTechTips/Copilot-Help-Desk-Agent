This is a step-by-step guide to creating and setting up an IT Help Desk Agent for your tenant.
First things first, let's start with the prerequisites that you need to begin this setup.
Step 1.
Verify licences and access.
Before you touch anything, confirm your tenant has the right licences; missing one will cause the silent failures later.
What you need
1. Microsoft 365 Business Standard or higher. - For Power Automate and Shared Mailbox
2. Copilot Studio Licences - standalone or through Microsoft 365 Copilot
3. Power Automate ( Per-user or per-flow ) - to send an email on the ticket submission
4. Exchange Online - required for the shared mailbox to work.
