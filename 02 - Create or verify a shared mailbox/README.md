Lets Create the Shared mailbox which will be used by the Agent
This is where IT tickets will be delivered

A shared mailbox is the destination for all tickets submitted through the agent. It does not need its own licences - It is accessed by your It team`s person accounts

Let's create the shared mailbox.
1. Go to https://admin.cloud.microsoft/exchange#/mailboxes
2. Click Recipients
3. Select Mailboxes from drop down menu.
4. +Add a Shared mailbox
5. Enter a display name, e.g. IT Help Desk
